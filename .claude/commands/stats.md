# /stats - 博客统计报告

生成详细的博客统计报告，包括文章数量、分类分布、标签使用等。

## 执行步骤

### 1. 统计文章总数

```bash
find source/_posts -name "*.md" -type f | wc -l
```

### 2. 按分类统计

```bash
# AI分类
find source/_posts/AI -name "*.md" -type f | wc -l

# AI子分类
find source/_posts/AI/computer-vision -name "*.md" -type f 2>/dev/null | wc -l
find source/_posts/AI/tensorflow -name "*.md" -type f 2>/dev/null | wc -l
find source/_posts/AI/tvm -name "*.md" -type f 2>/dev/null | wc -l

# system-design分类
find source/_posts/system-design -name "*.md" -type f | wc -l

# other分类
find source/_posts/other -name "*.md" -type f | wc -l
```

### 3. 统计总字数

```bash
find source/_posts -name "*.md" -type f -exec wc -w {} + | tail -1
```

### 4. 最近更新的文章

```bash
# 最近30天更新的文章
find source/_posts -name "*.md" -type f -mtime -30 -exec ls -lh {} \; | sort -k6,7
```

### 5. 提取所有标签

分析所有文章的Front Matter，提取标签并统计使用频率。

### 6. 文章长度分析

- 最长的文章
- 最短的文章
- 平均文章长度

### 7. 图表文件统计

```bash
find source/diagrams -name "*.excalidraw" -o -name "*.mmd" | wc -l
```
