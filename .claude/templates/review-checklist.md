# 文章审查清单

审查文章时，请检查以下 8 个方面：

## 1. Front Matter 完整性
- [ ] title 字段存在且非空
- [ ] date 字段格式正确（YYYY-MM-DD）
- [ ] categories 字段存在且符合规范
- [ ] tags 字段至少有 2 个标签

## 2. 内容结构
- [ ] 有清晰的引言
- [ ] 标题层级正确（# 只用一次）
- [ ] 使用 ## 和 ### 组织内容
- [ ] 有明确的总结

## 3. 代码质量
- [ ] 所有代码块都指定了语言
- [ ] 代码示例完整可运行
- [ ] 复杂代码有必要的注释

## 4. 格式规范
- [ ] 中英文之间有空格
- [ ] 标点符号使用正确
- [ ] 链接格式正确 [文字](URL)

## 5. 技术准确性
- [ ] 技术术语使用一致
- [ ] 概念解释准确
- [ ] 引用来源标注清楚

## 6. SEO 优化
- [ ] 标题包含关键词
- [ ] 至少 3 个相关标签
- [ ] 文章长度适中（建议 800 字以上）

## 7. 可读性
- [ ] 语言流畅，逻辑清晰
- [ ] 段落长度适中
- [ ] 有适当的示例和图表

## 8. 链接检查
- [ ] 所有链接可访问
- [ ] 内部链接路径正确
