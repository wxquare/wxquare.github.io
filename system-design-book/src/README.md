# 程序员系统设计与面试指南

这是一本**手册型电子书**，按**先工程、后面试**的顺序组织内容：先讲系统设计、基础设施、计算机基础与工程实践，再在各章末尾收束为「本章面试题与追问」。全书不做题库式拼接，而是用统一主线串起知识、方案与表达。

## 三条阅读路线

1. **主线必读**：第 1–9 章，覆盖导论、方法论、中间件、可靠性与系统设计面试主干。
2. **支撑扩展**：第 10–15 章，补足操作系统、网络、Shell、Python、C++、Go 等工程基础。
3. **编码补充**：第 16–21 章，作为 LeetCode 与编码模式的独立补充，不降格为附录。

## 使用方式

- 想快速建立系统设计主干：先读主线必读，再按需回看支撑扩展。
- 想兼顾工作与面试：按目录顺序阅读，保留章末的追问区作为复盘入口。
- 想做长期查阅：将本书当作工程手册使用，结合源博客持续补充各章内容。

## 本地构建

需安装 [mdBook](https://github.com/rust-lang/mdBook) 与可选的 `mdbook-mermaid`（与 `ai-book` / `ecommerce-book` 一致）。

```bash
cd system-design-book
mdbook build
mdbook serve
```
