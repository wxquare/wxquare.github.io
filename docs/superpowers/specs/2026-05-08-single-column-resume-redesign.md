# 单栏简历重写设计说明

## 目标

重写两份一页中文简历：

1. 后端架构版：面向后端架构、电商核心系统、交易平台、平台化岗位。
2. AI Agent 版：面向 AI Agent 工程、Agent 应用后端、AI 平台后端岗位。

新版保留单栏，不使用双栏。重点不是压缩已有内容，而是重新组织叙事、成果和排版层级，让 8 年经验、腾讯 + Shopee 背景、系统设计沉淀和正在写的书籍共同服务于候选人定位。

## 当前问题

上一版的主要问题：

- 内容像项目流水账，所有经历都用同一种 bullet 表达，主次不清。
- 单栏宽度没有问题，但文字层级单薄，阅读节奏差。
- 页面上半部拥挤，下半部留白，空间没有被结构化使用。
- 缺少高级候选人的职业摘要、关键成果和能力画像。
- 没有充分利用仓库中的书籍、博客和面试材料作为作品集证据。
- 两份简历只是“项目顺序不同”，而不是围绕不同岗位重新组织叙事。

## 版式方向

仍然使用单栏 A4 一页，但采用更清晰的信息架构：

1. 页眉：姓名、联系方式、求职方向。
2. 职业摘要：2 行以内，直接说明候选人定位。
3. 关键成果：一行指标组，突出最硬的量化成果。
4. 工作经历：按公司和主题组织，Shopee 为主体，腾讯为补充。
5. 技术输出：把三本书和博客沉淀作为作品集证据。
6. 教育背景：放底部，压缩展示。

排版上使用：

- 更合理的页边距和正文宽度。
- 项目标题、公司标题、摘要块、指标条形成视觉层级。
- 少量灰色背景或边框用于摘要和指标，不做花哨设计。
- 每个项目块使用“主题标题 + 一句背景 + 2 条强成果”。
- 删除平均铺开的弱 bullet，减少重复技术名词。

## 后端架构版叙事

定位：`后端架构 / 电商核心系统`

职业摘要重点：

- 8 年后端架构经验，腾讯广告 + Shopee 数字电商。
- 主导多品类商品、库存、计价、OTA 与稳定性体系建设。
- 擅长复杂业务平台化、高并发交易链路、DDD、一致性、资金安全和系统可靠性。

关键成果候选：

- 10+ 品类统一平台。
- 日均 5000 万+ 计价调用。
- 秒杀峰值 20,000 QPS。
- 资损事故 3-5 次/年降至 0。
- 新品类接入 2 周降至 2 天。
- 万级 SKU 批量操作从数小时降至分钟级。
- 2-3 个月交付 OTA 单品类。

工作经历组织：

### Shopee

主题一：电商交易与资金安全架构

- 合并计价中心、价格快照、支付校验、灰度迁移、性能优化。
- 体现 DDD、场景驱动、快照一致性、资损控制、P99 优化。

主题二：多品类商品供给平台化

- 合并商品模型、运营后台、供应商接入、异步上架、批量操作框架。
- 体现跨品类抽象、状态机、策略模式、Outbox、Saga、审计追溯。

主题三：库存与高并发履约

- 合并统一库存、Redis Lua、券码池、供应商 Booking、对账。
- 体现高并发、数据一致性、降级和可恢复。

主题四：复杂业务交付与稳定性

- 压缩 OTA 和稳定性治理，作为综合交付能力证明。

### 腾讯

压缩为两条：

- 微信广告调价数据流与营销中台。
- AI 短视频原生广告植入和 TVM 优化。

## AI Agent 版叙事

定位：`AI Agent 工程 / 后端架构`

职业摘要重点：

- 8 年后端工程与架构经验，正在系统化沉淀 AI Agent 工程实践。
- 关注 Agent Runtime、Tool Runtime、RAG、MCP、状态机、多 Agent 编排、Evals、Guardrails 与可观测性。
- 优势不是 demo，而是把 Agent 放进真实生产系统。

关键成果候选：

- DoD Agent 自动处理 65%+ 告警。
- 告警响应 15 分钟降至 <2 分钟。
- 告警噪音减少 70%。
- 10+ MCP 工具接入。
- 电商系统日均 5000 万+ 计价调用、20,000 QPS 库存。
- AI Agent 工程实践书覆盖 20 章。

工作经历组织：

### Shopee

主题一：生产级 AI 告警处理 Agent

- 突出状态机 + ReACT、RAG、MCP 工具、风险分级、审计、trace、人工确认。
- 用指标证明实际价值。

主题二：Agent 工程底座能力

- 把电商系统中的状态机、异步任务、幂等、灰度、审计、监控、对账、降级提炼成 Agent 生产化能力。

主题三：电商核心系统架构

- 压缩商品、库存、计价为后端架构可信度，不再抢过 DoD Agent 主线。

### 腾讯

突出 AI 广告植入、TVM 推理优化，以及广告工程经验。

### 技术输出

突出：

- 《AI Agent 工程实践：从大模型基础到生产级智能体系统》
- DoD Agent 案例、企业知识助手、可观测 Coding Agent、PKM Agent
- 电商系统架构设计与系统设计面试指南作为后端架构底座
